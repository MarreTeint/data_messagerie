## Description

A simple UI to display messages structured with JSON Schema.

## How to run

1. Clone the repository 

```bash
git clone https://github.com/MarreTeint/data_messagerie.git
```

2. Install the dependencies

```bash
npm install
```

3. Run the project

```bash
npm run dev
```
